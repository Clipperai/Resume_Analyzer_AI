# 📄 Resume Analyzer AI (Streamlit + Groq)

A smart AI-powered Resume Analyzer that extracts text from resumes (PDF/DOCX) and provides instant AI feedback like summary, skills, and improvements.

---

## 🚀 Features
- 📤 Upload Resume (PDF / DOCX)
- 🤖 AI-powered analysis using Groq LLM
- 🧠 Extracts:
  - Resume summary
  - Key skills
  - Improvement suggestions
- ⚡ Fast Streamlit UI

---

## 🛠️ Tech Stack
- Python
- Streamlit
- PyPDF2 / pypdf
- python-docx
- Groq API
