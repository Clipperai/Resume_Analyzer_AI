# 📄 Resume Analyzer AI (Streamlit + Groq)

A smart AI-powered Resume Analyzer that extracts text from resumes (PDF/DOCX) and provides instant AI feedback like summary, skills, and improvements.

---

## 🚀 Features
- 📤 Upload Resume (PDF / DOCX)
- 🤖 AI-powered analysis using Groq LLM
- 🧠 Extracts:
  - Resume summary
  - Key skills
  - Improvement suggestions
- ⚡ Fast Streamlit UI

---

## 🛠️ Tech Stack
- Python
- Streamlit
- PyPDF2 / pypdf
- python-docx
- Groq API

---

## TRY IT NOW 
- Link:- https://resume-ai1.streamlit.app/

---
## ⚙️ How It Works
1. Upload resume (PDF/DOCX)
2. Extract text using PyPDF2 / python-docx
3. Send text to Groq LLM
4. Get structured AI feedback
5. View analysis + history in UI
